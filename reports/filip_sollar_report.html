Zip Export functionality in the semestral project

My task was to create java class which exports europass xml CVs stored in database to ZIP Archive.

First I have created java class which exported CVs using ZipOutputStream from java.util package

Later I have migrated to Zip4j library which improved scalability by adding multiple methods for manipulation Zip archives.

Example of usage

  • java -jar europasstoolkit-1.0-SNAPSHOT.jar export_zip [name_of_xml_in_DB | multiple_names]

  • option: -a (export all XML files in database)

  • option: -o (sets output file name, by default result.zip is used)
