Store functionality in semestral project

My task was to create java class which stores europass cvs in format XML to native XML database. Program works as command line tool with option store. Also you have to add path or paths to your cvc in format XML. We chose BaseX native database which is suitable for not huge amount of data. Connection to database is provided by XQJ api in java. All settings for connection are stored in dbSetup.properties. Default name of database is europassDB which can be changed in dbSetup.properties. Root element of db is tag <europasses>. Every europass cv is stored as child element of tag <europass name=example> Name of xml has to be unique.

Example of usage

  • java -jar europasstoolkit-1.0-SNAPSHOT.jar store {path_to_europass}/europass-sample1.xml

  • java -jar europasstoolkit-1.0-SNAPSHOT.jar store {path_to_europass}/europass-sample1.xml {path_to_europass}/europass-sample2.xml
